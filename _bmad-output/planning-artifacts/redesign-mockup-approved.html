AWS Network Atlas active

Scans AWS accounts read-only and renders an accurate, interactive network & security diagram — diffable plain text that lives in git alongside your code.

TypeScriptAWS SDKReact
github ↗
full write-up

Point it at one or more AWS profiles and it stitches a picture of your estate across accounts and regions: VPCs, Transit Gateways, peering, VPN, Direct Connect, load balancers, edge/DNS, IAM, KMS, WAF, and more.

Every snapshot is plain text, so the whole diagram is diffable and lives in git alongside your code. A synthetic demo estate ships with the repo — open the viewer with no AWS account and no scan.

Built to solve the problem of wanting an actively maintained, accurate AWS estate diagram without granting 3rd-party access to your account.
